# Todo Extension

## Provenance

- **Source**: https://github.com/badlogic/pi-mono
- **Path**: `packages/coding-agent/examples/extensions/todo.ts`
- **Commit**: d9cfa115bed37c6036734ec3ab5ad1cbbda6f36a

## Description

A todo extension that demonstrates state management via session entries. It provides:

- A `todo` tool for the LLM to manage todos (list, add, toggle, clear)
- A `/todos` command for users to view the todo list in an interactive UI
- State stored in tool result details (not external files) for proper session branching
